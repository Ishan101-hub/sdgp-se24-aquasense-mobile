# AquaSense Mobile

AquaSense is an IoT-based Smart Multi-Zone Water Monitoring System designed to provide real-time water flow monitoring, leak detection, and remote valve control.

The system integrates ESP32-based IoT devices, MQTT messaging via HiveMQ Cloud, a Python-based backend for data processing and storage, and a Flutter mobile application for real-time visualization and control.

The architecture follows a modular and layered design, ensuring separation of concerns between device firmware, backend services, and the mobile application. The system supports real-time data streaming, event logging, automated leak detection, and scalable deployment using CI/CD pipelines.
