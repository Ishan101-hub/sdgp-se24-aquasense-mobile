# Code of Conduct

AquaSense is a team project built through collaboration, respect, and shared responsibility.

## Expected Behaviour

- Communicate respectfully.
- Give constructive feedback.
- Credit team members for their contributions.
- Keep discussions focused on improving the project.
- Help teammates understand technical decisions.

## Unacceptable Behaviour

- Harassment or personal attacks.
- Removing another member's work without discussion.
- Misrepresenting individual contributions.
- Committing secrets or unsafe code knowingly.

## Conflict Resolution

If disagreements occur, discuss the technical reason, compare alternatives, and choose the solution that best supports maintainability, reliability, and user safety.
