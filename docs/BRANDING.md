# AquaSense Branding Guide

## Project Name

AquaSense

## Taglines

- Smart water awareness, one zone at a time.
- Real-time monitoring. Reliable conservation.
- Detect leaks before they become losses.
- Intelligent water control for modern homes.

## Tone

Professional, clean, sustainability-focused, engineering-driven.

## Suggested Color Palette

| Color | Use |
|---|---|
| Navy Blue | Primary brand / buttons / headers |
| Aqua Blue | Water identity / highlights |
| White | Clean UI background |
| Light Blue | Cards / secondary backgrounds |
| Red | Leak alerts / warnings |
| Green | Safe status / valve open |

## Visual Themes

- Water droplets
- Flow lines
- Circular gauges
- Zone cards
- Dashboard analytics
- IoT connectivity
