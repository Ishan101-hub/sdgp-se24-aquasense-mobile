## Long Portfolio Description

AquaSense is an IoT-based smart multi-zone water management system developed to reduce domestic water wastage. The system uses ESP32 microcontrollers, water flow sensors, solenoid valves, and MQTT communication to monitor real-time usage and detect leaks at zone level. A FastAPI backend processes incoming sensor data, stores it in Supabase PostgreSQL, performs aggregation and leak detection, and exposes APIs to a Flutter mobile/web interface. I contributed as the Project Lead & IoT Systems Engineer, focusing on IoT communication, MQTT integration, device testing, system integration, repository management, and production deployment.
